🚀 User Management and Backup Automation (Linux)

📌 Project Overview

This project is a Python-based automation tool designed to manage Linux users and perform directory backups efficiently. It is deployed and tested on an AWS EC2 instance and demonstrates practical DevOps and system administration skills.

⚙️ Features

👤 User Management

Add new users

Delete existing users

Modify user passwords

👥 Group Management

Create new groups

Add users to groups

📦 Backup System

Backup any directory

Compress into .tar.gz format

Timestamp-based backup naming

📝 Logging

All actions are logged with date and time

🛠️ Technologies Used

Python 3

Linux (Ubuntu)

AWS EC2

Shell Commands (via subprocess)

Git & GitHub
